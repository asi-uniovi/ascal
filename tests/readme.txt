-The test is based on comparing the sequence of allocations after applying autoscaling to a set of problems.

- The reference allocation sequences are stored in the "solutions" directory.
They are generated by running test_gen/gen_solutions.py and copying the output files from the
newly created "new_solutions" directory into "solutions".

- The "solutions" directory also contains a log file per problem that helps identify the source of discrepancies.

- To run the test, execute pytest .\tests\test_gen\test_problems.py from the project's root directory.

- To run the test and perform a coverage analysis with an HTML report, run from the project's root directory:
        pytest --cov=ascal --cov-report=html --cov-report=term
